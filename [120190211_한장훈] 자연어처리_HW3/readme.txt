input.txt를 기준으로 SMASH를 실행하여 result.txt파일을 만들었습니다.
작성한 코드는 result.txt 파일과 train.txt파일을 이용하여 확률을 계산하고 HMM을 구현합니다.
result.txt와 train.txt는 hmm.py와 같은 위치입니다.
결과는 output.txt파일로 나오게 됩니다.

형태소 분석기 SMASH파일은 용량이 너무커서 입력 텍스트와 출력텍스트인 input.txt result.txt만 첨부하였습니다.
스무딩은 laplace 스무딩 방법을 사용하였습니다.
